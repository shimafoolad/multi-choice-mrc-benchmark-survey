# 🤝 Contributing

Thank you for your interest in improving this survey repository! Since MRC is a fast-moving field, community contributions help keep this resource accurate and up-to-date.

---

## How to Contribute

### 🐛 Report a Broken Link
Open an **Issue** with the label `broken-link` and include:
- The dataset name
- The broken URL
- A working alternative URL (if you have one)

### 📈 Update a SOTA Result
Open an **Issue** with the label `sota-update` and include:
- The dataset name
- The new SOTA result (%) and metric
- The model name and paper/link
- Whether the dataset is now "solved" (model ≥ human)

### ➕ Add a Missing Dataset
Open a **Pull Request** that adds a row to the relevant table file (`datasets_part1.md` or `datasets_part2.md`) with:
- Year, dataset name, vocab size, question size, context size
- Data source, metric, human performance, SOTA performance
- Solved status and dataset link

### ✏️ Fix a Typo or Clarify Text
Just open a Pull Request — small fixes are always welcome.

---

## Guidelines

- Keep table formatting consistent (see existing files as reference).
- Cite the source of any new SOTA result (arXiv ID or ACL Anthology link preferred).
- For new datasets, only include those published in peer-reviewed venues or widely cited preprints.

---

We appreciate every contribution, large or small!
